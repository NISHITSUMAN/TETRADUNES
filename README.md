# ⛱️ Tetradunes

**Tetradunes** is a fun and fast-paced terminal-based game inspired by the classic *Sandtrix*. Your goal is to skillfully manage falling blocks of various shapes and align them into complete rows to score points. But there’s a twist — sandy terrain introduces new dynamics to the classic falling-block formula.

> ⚠️ Some parts of this README were created using AI assistance.  
> 🎮 Inspired by Sandtrix: [Play the original here](https://www.sandtrix.net/)

---

## 🕹️ Gameplay Overview

- **🎯 Objective**: Fit falling blocks into horizontal rows to clear them and score points.
- **💥 Game Over**: The game ends when blocks stack up to the top of the board.
- **🌪️ Unique Twist**: Sandy terrain alters how blocks interact and move, making for an extra strategic challenge.

---

## 🚀 How to Play

1. **Start the Game**
   - Click one "Play Game"

2. **Control the Blocks**
   - `←` Left Arrow: Move block left  
   - `→` Right Arrow: Move block right  
   - `↑` Up Arrow: Rotate block  
   - `↓` Down Arrow: Increase fall speed

3. **Clear Rows**
   - Strategically stack and rotate blocks to form complete horizontal lines.  
   - Each cleared row increases your score.

4. **Adapt to the Sand**
   - The game physics may differ due to sand behavior. Plan movements carefully.
